# ATM Simulation in C++

This project simulates a simple ATM system using C++.  
It includes PIN authentication, balance checking, cash withdrawal, and deposit functionalities.

## 📊 Project Features
- **PIN Authentication**
- **Check Balance**
- **Withdraw Cash**
- **Deposit Cash**
- **Error Handling** for invalid amounts or insufficient funds
